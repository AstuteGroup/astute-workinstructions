Market Offer Uploads. The Specialist takes files or emails forwarded from excessinc@astutegroup.com or offers@astutegroup.com to request@orangetsunami.com and formats the data into the  ​csv icon Market Offer Line Import Template.csv or saved into the terminal for immediate action.
            Key fields to be mapped are:
Chuboe_Offer_ID[Value]         : generated by OT. User may provide this, but if not then leave blank
Chuboe_MPN                          : Manufacturer’s part number
Qty
Secondary Fields
Chuboe_MFR_Text                            : Free text field to include the manufacturer
Chuboe_Date_Code               
Description                              : This is used for the part description if available
Chuboe_CPC                           : If a customer includes their internal part number then it is populated here
PriceEntered                           : cost if provided
Chuboe_MOQ             : Minimum order quantity if provided
Chuboe_SPQ                           : Standard pack size (mult) if provided
The file must be generated in a .csv format and saved. All header fields must be included even if there is no data populated in the column.
